# 🎬 Movie Recommender System

A content-based movie recommendation web application built with Python, NLP, 
and Machine Learning. Enter any movie title and get 5 similar movie 
recommendations instantly — powered by cosine similarity on TMDB metadata.

🔗 **Live Demo:** [Coming Soon / Add Streamlit Cloud link here]

---

## ✨ Features

- 🔍 Search from 4,800+ movies from the TMDB 5000 dataset
- 🤖 NLP-powered content-based filtering using Bag of Words
- ⚡ Real-time recommendations via precomputed cosine similarity
- 🌐 Interactive web UI built with Streamlit
- 🏷️ Tag-based feature engineering from genres, cast, crew, keywords & overview
